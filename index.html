01 — Core Concepts

Why ResNet Changed Everything

Before ResNet (2015), deeper networks often performed worse than shallow ones — a counter-intuitive result caused by the degradation problem and vanishing gradients.

📉

Vanishing Gradients

As backprop flows through many layers, gradients shrink exponentially. Early layers receive near-zero updates, preventing learning in deep networks.

⤵️

Degradation Problem

Adding more layers to a plain network causes accuracy to saturate and then degrade — even on training data. This is not overfitting; it's an optimization failure.

➕

Residual Learning

Instead of learning H(x) directly, a block learns F(x) = H(x) − x (the residual). The output becomes F(x) + x via a skip connection — identity shortcut.

🔗

Skip Connections

Shortcuts bypass one or more layers, adding the input directly to the output. Gradients can now flow freely through the shortcut path, solving vanishing gradients.

🏗️

Identity Mapping

If extra layers are unnecessary, the network can simply learn F(x) = 0, making the block behave as an identity. This makes depth never harmful.

🏆

ILSVRC 2015 Winner

ResNet-152 (152 layers!) achieved 3.57% top-5 error on ImageNet — surpassing human performance (~5.1%) for the first time.
